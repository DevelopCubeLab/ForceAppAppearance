# ForceAppAppearance
iOS Tweaks force app dark or light appearance  
**The App itself does not support dark appearance, and the injected plug-in has no effect.**  

iOS 插件 强制App使用深色或者浅色模式，解决某些app不提供切换深色/浅色模式的开关。  
**App本身不支持深色模式的，注入插件也无效。**

## Usage
使用`TrollFools`或者其他签名工具注入。  
Inject using `TrollFools` or other signature tools.  

## Build
1. Download [Theos](https://theos.dev/)
2. run `make package` or `make stage` if you want to build rootless version run `make package THEOS_PACKAGE_SCHEME=rootless`

Dark appearance in [Branch](https://github.com/DevelopCubeLab/ForceAppAppearance/tree/dark)  
Light appearance in [Branch](https://github.com/DevelopCubeLab/ForceAppAppearance/tree/light)
